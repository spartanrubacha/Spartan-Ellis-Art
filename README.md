# SPARTAN ELLIS ART | Digital Gallery

This is a custom-built, interactive web gallery designed to showcase the paintings of Spartan Ellis. 

## 🎨 The Experience
The gallery features a dynamic "drifting" interface where painting icons float in a randomized field. 
- **Interactive Exploration:** Hover over an icon to summon the painting from the void.
- **Sidebar Alignment:** When a painting is active, icons align vertically behind the navigation for a clean viewing experience.
- **Micro-Interactions:** Includes a custom glass-lens zoom tool for inspecting brushwork and texture.

## 🛠️ Technical Stack
- **Frontend:** Vanilla HTML5, CSS3 (Flexbox/Grid), and JavaScript.
- **Animations:** CSS Transitions with custom cubic-bezier curves for a "prestige" feel.
- **Logic:** Custom Poisson-disk-inspired placement for non-overlapping icon generation.
